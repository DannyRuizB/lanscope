Advanced options
Ports
Top = nmap's most common N TCP ports. Range = explicit list.
NSE scripts
Run nmap NSE scripts during the TCP scan. Default = the -sC set (banners, http-title, ssh-hostkey…). Safe = broader, still classified non-intrusive. Both unchecked = no scripts. Adds time to the scan.
Host discovery
When checked, nmap treats every host in the CIDR as up and runs no probes — useful when ICMP/SYN are blocked. Other checkboxes below are ignored.
Pick which probes nmap uses to decide who's alive. Nothing checked = nmap defaults (echo + TCP SYN to 443 + TCP ACK to 80 + ICMP timestamp, plus ARP on local LAN). Checking some restricts to only those.
